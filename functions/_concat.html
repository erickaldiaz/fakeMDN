El método _concat() se usa para unir dos o más arrays. Este método no cambia los arrays existentes, sino que devuelve un nuevo array.

Sintaxis

_concat(arrays1, array2, ...arrayN)

Parámetros

array
Los arrays que se unirán para conformar el nuevo array a devolver.

Valor de retorno

Devuelve un nuevo array con la concatenación de todos los pasados por parámetros.

Ejemplo

Concatenando dos arrays

            
const alpha = ['a', 'b', 'c'];

const numeric = [1, 2, 3];

_concat(alpha, numeric);
        
>>  ['a', 'b', 'c', 1, 2, 3]
