El método _max() devuelve el valor máximo de un array. Los elementos del array pueden ser number o string.

Sintaxis

_max(array)

Parámetros

array
El array de cuyos elementos se devolverá el máximo.

Valor de retorno

Devuelve un number o un string, o undefined en caso de que el array esté vacío.

Ejemplo

Averiguar el valor más alto de un array de números

            
const lista = ['a', '1', 'j'];

fakeArrayMax(lista);

>>> j
