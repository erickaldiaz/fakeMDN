El método _sum() calcula la suma de los valores de un array.

Sintaxis

_sum(array)

Parámetros

array
El array cuyos valores serán sumados.

Valor de retorno

Devuelve un número.

Ejemplo

Sumar los valores de un array

            
const numeros= [2, 5, 9, 15];

_sum(numeros);
              
>> 31
