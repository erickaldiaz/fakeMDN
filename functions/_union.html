A patir de dos arrays, el método _union() retorna un nuevo array con la unión de los valores de ambos, sin repeticiones y preservando el orden de los elementos, comenzando desde el primer elemento del primer array pasado por parámetro.

Sintaxis

_union(arrayOne, arrayTwo)

Parámetros

arrays
Los arrays que se unirán para conformar el nuevo array a devolver.

Valor de retorno

Devuelve un nuevo array.

Ejemplo

Unir dos arrays de números sin repeticiones

            
const firstArray = [1, 2, 3];

const secondArray = [3, 4, 5];
              
_union(firstArray, secondArray);
              
>> [1, 2, 3, 4, 5]
