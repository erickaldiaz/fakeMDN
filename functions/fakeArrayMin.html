El método fakeArrayMin() devuelve el valor mínimo de un array. Los elementos del array pueden ser number o string.

Sintaxis

fakeArrayMin(array)

Parámetros

array
El array de cuyos elementos se devolverá el mínimo.

Valor de retorno

Devuelve un number o un string, o undefined en caso de que el array esté vacío.

Ejemplo

Averiguar el valor más bajo de un array de números

            
    const numeros= [1, 4, 9, 88, -44, 109, 8];

    fakeArrayMin(numeros);
              
    >> -44

Averiguar el valor más bajo de un array de números y strings

            
    const lista = ['a', '1', 'j'];

    fakeArrayMin(lista);

    >>> 1
