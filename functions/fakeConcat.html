El método fakeConcat() se usa para unir dos o más arrays. Este método no cambia los arrays existentes, sino que devuelve un nuevo array.

Sintaxis

fakeConcat(arrayOne, arrayTwo)

Parámetros

arrays
Los arrays a concatenar

Valor de retorno

Devuelve un nuevo array con ambos arrays de entrada concatenados.

Ejemplo

Concatenando dos arrays

              
    const alpha = ['a', 'b', 'c'];
    const numeric = [1, 2, 3];
    
    fakeConcat(alpha, numeric);
              
    >>  ['a', 'b', 'c', 1, 2, 3]
