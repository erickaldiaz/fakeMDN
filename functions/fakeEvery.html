El método fakeEvery() devuelve un booleano, true si todos los elementos en el arreglo pasan la condición implementada por la función dada y false si alguno no la cumple.

Sintaxis

fakeFilter(array, callback(currentValue))

Parámetros

array
El array sobre el que se ha llamado every.
callback
Función para probar cada elemento.
currentValue
El elemento actual del array que está siendo procesado.

Valor de retorno

true si la función de devolución de llamada devuelve un valor de truthy para cada elemento de matriz; de lo contrario, false.

Ejemplo

Probando el tamaño de todos los elementos de un arreglo

function esSuficientementeGrande(elemento) { return elemento >= 10; }

const filtrados1 = [];

fakeEvery([12, 5, 8, 130, 44], esSuficientementeGrande); //false

fakeEvery([12, 54, 18, 130, 44], esSuficientementeGrande); //true
