El método fakeFind() devuelve el primer elemento del array que cumple con la condición implementada por una función que se pasa como argumento.

Sintaxis

fakeFind(array, callback)

Parámetros

array
El array sobre cuyos elementos se comprueba si cumplen con la condición provista por la función.
callback
Función que se va a aplicar a los elementos hasta encontrar alguno que la cumpla.

Valor de retorno

Devuelve un elemento del array o undefined en caso de que ninguno cumpla con la condición.

Ejemplo

Encontrar un elemento en el array mayor a 15

let listOfNumbers = [5, 8, 30, 84];

const isOver15 = (x) => x > 15 ? true : false;

fakeFind(listOfNumbers, isOver15);

>> 30
