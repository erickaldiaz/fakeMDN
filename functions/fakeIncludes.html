El método fakeIncludes() determina si un array tiene un determinado elemento, devuelve true o false según corresponda.

Sintaxis

fakeIncludes(array, callback)

Parámetros

array
El array sobre cuyos elementos se comprueba si cumplen con la condición provista por la función.
callback
Función que se va a aplicar a los elementos hasta encontrar alguno que la cumpla.

Valor de retorno

Devuelve true si encuentra el elemento que buscaba y false si no lo encuentra en el array.

Ejemplo

Usando Includes

let array = 'To be, or not to be, that is the question.';

const element= 'question'

fakeinclude(array, element);

>> True
