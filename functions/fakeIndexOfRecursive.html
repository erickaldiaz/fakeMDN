El método fakeIndexOfRecursive() devuelve el primer índice en el que se puede encontrar un elemento dado en el array, o devuelve -1 si el elemento no está presente. Además, es implementado utilizando recursividad.

Sintaxis

fakeIndexOfRecursive(array, element)

Parámetros

array
El array en el que se va a buscar el elemento.
element
El elemento que va a ser buscado en el array.

Valor de retorno

Se devuelve un número, correspondiente al primer índice del array en el que se encuentra el elemento, o -1 si el elemento no está presente.

Ejemplo

Encontrar valores en un array

						
		const numbers = [2, 3, 9, 8, 67, 7];
		fakeIndexOfRecursive(numbers, 67);
		>> 4
