El método fakeLastIndexOf() devuelve el último índice en el que un elemento puede encontrarse o devuelve -1 si el elemento no se encontrara

Sintaxis

fakeLastIndexOf(element)

Parámetros

Element
Elemento del cuál indicará el indice.

Valor de retorno

Devuelve el indice del elemento o -1 en caso de no encontrar el elemento solicitado.

Ejemplo

Buscando elementos en un array

              

  let animales = ['perro', 'tigre', 'pinguino', 'perro'];

  console.log(animales.fakeLastIndexOf('perro'));
  // expected output: 3

  console.log(animals.lastIndexOf('tigre'));
  // expected output: 1
