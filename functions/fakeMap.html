El método fakeMap() crea un nuevo array con los resultados de la llamada a la función indicada aplicados a cada uno de sus elementos.

Sintaxis

fakeMap(array, callback(currentValue))

Parámetros

array
El array sobre cuyos elementos se comprueba si cumplen con la condición provista por la función.
callback
Función que se va a aplicar a los elementos del array.
currentValue
El elemento actual del array que se está procesando.

Valor de retorno

Devuelve un nuevo array donde cada elemento tiene aplicada la función callback.

Ejemplo

Procesar un array de números aplicándoles la raíz cuadrada

const numeros= [1, 4, 9];

fakeMakp(numeros, Math.sqrt);

>> [1, 2, 3]

Mapear un array de números usando una función con un argumento

const numeros = [1, 4, 9];

const dobles = fakeMap(numeros, (function(num) { return num * 2; });

>>> dobles = [2, 8, 18]
