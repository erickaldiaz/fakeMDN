El método fakeReduce() aplica una función a un acumulador y a cada valor de un array (de izquierda a derecha) para reducirlo a un único valor.

Sintaxis

fakeReduce(array, callback [,initialValue])

Parámetros

array
El array sobre cuyos elementos se quiere reducir a uno aplicándoles una función.
callback
Función que se va a aplicar a los elementos del array.
initialValue
Objeto a usar como primer argumento en la primera llamada de la función callback.

Valor de retorno

Depende del tipo de dato que contenga el array y del tipo de dato del valor inicial que se pase como argumento.

Ejemplo

Multiplicación de los números de un array

            
  let listOfNumbers = [5, 6, 2, 3];

  const multiply = (x, y) => x * y;
            
  fakeReduce(listOfNumbers, multiply);
            
  >> 180

Generar un array con los valores de una propiedad de varios objetos

            
  objetos = [
			{a: 5, b: 10, c: 15},
			{a: 1, b: 2, c: 3},
			{a: 4, b: 8, c: 12}
			];

  objetos.reduce((acc, cur) => {acc.push(cur.b); return acc}, []);
            
  >> [10, 2, 8]
