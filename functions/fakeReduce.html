El método fakeReduce() aplica una función a un acumulador y a cada valor de un array (de izquierda a derecha) para reducirlo a un único valor.

Sintaxis

fakeReduce(array, callback)

Parámetros

array
El array sobre cuyos elementos se quiere reducir a uno aplicándoles una función.
callback
Función que se va a aplicar a los elementos del array.

Valor de retorno

Depende del tipo de dato que contenga el array, puede ser un Number, un String o un array.

Ejemplo

Multiplicación de los números de un array

let listOfNumbers = [5, 6, 2, 3];

const multiply = (x, y) => x * y;

fakeReduce(listOfNumbers, multiply);

>> 180
