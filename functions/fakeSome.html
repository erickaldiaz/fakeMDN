El método fakeSome() comprueba si al menos un elemento del array cumple con la condición implementada por una función que se pasa como argumento.

Sintaxis

fakeSome(array, callback)

Parámetros

array
El array sobre cuyos elementos se comprueba si cumplen con la condición provista por la función.
callback
Función que se va a aplicar a los elementos hasta encontrar alguno que la cumpla.

Valor de retorno

Devuelve un booleano: true en caso de que algún elemento cumpla con la condición de la función o false en caso de que ninguno la cumpla.

Ejemplo

Comprobar si algún número del array es impar

let listOfNumbers = [10, 21, 30, 40];

const isOdd = = (x) => x % 2 != 0 ? true : false;

fakeSome(listOfNumbers, isOdd);

>> false
