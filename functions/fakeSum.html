El método fakeSum() calcula la suma de los valores de un array.

Sintaxis

fakeSum(array)

Parámetros

array
El array sobre cuyos valores serán sumados.

Valor de retorno

Devuelve un número.

Ejemplo

Sumar los valores de un array

              
    const numeros= [2, 5, 9, 15];

    fakeSum(numeros);
              
    >> 31
