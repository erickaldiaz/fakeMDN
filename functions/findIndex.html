El método _findIndex() devuelve el primer índice del primer elemento de un array que cumpla con la función de prueba proporcionada. En caso contrario devuelve -1.

Sintaxis

_findIndex(array, callback)

Parámetros

array
El array cuyos elementos serán pasados como parámetros a la función provista.
callback
Función a ejecutar para cada uno de los elementos del array.

Valor de retorno

Devuelve el índice del primer elemento que cumpla con la función, o -1 en caso de que ningún elemento la cumpla.

Ejemplo

Encontrar el índice del primer elemento igual a otro

              
const mascotas = ['perro', 'gato', 'canario', 'pez', 'tortuga', 'hurón'];

const esGato = mascota => mascota === 'gato';

_findIndex(mascotas, esGato);
              
>> 1
