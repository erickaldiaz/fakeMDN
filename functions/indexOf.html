El método _indexOf() devuelve el primer índice en el que se puede encontrar un elemento dado en el array, o devuelve -1 si el elemento no está presente.

Sintaxis

_indexOf(array, element)

Parámetros

array
El array en el que se buscará el elemento.
element
El elemento que va a ser buscado en el array.

Valor de retorno

Se devuelve un número, correspondiente al primer índice del array en el que se encuentra el elemento, o -1 si el elemento no está presente.

Ejemplo

Encontrar valores en un array

              
let array = [2, 9, 9];

_indexOf(array, 2));

>> 0
