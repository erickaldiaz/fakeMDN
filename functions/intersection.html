A patir de dos arrays, el método _intersection() retorna un nuevo array con la intersección de los valores de ambos, sin repeticiones y preservando el orden de los elementos, comenzando desde el primer elemento del primer array pasado por parámetro.

Sintaxis

arrayOne._intersection(arrayTwo)

Parámetros

arrayTwo
El array con el que será comparado el array original.

Valor de retorno

Devuelve un nuevo array con la intersección.

Ejemplo

Intersectar dos arrays de números sin repeticiones

            
const firstArray = [1, 2, 3, 5];

const secondArray = [2, 4, 5, 7];

firstArray._intersection(secondArray);

>> [2, 5]
