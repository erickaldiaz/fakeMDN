El método _lastIndexOf() devuelve el último índice en el que un elemento puede encontrarse, o devuelve -1 si el elemento no se encontrara.

Sintaxis

array._lastIndexOf(element)

Parámetros

element
Elemento del cual se indicará el último índice en el que puede encontrarse, si está presente.

Valor de retorno

Devuelve el indice del elemento o -1 en caso de no encontrar el elemento solicitado.

Ejemplo

Buscando elementos en un array

              
let animales = ['perro', 'tigre', 'pinguino', 'perro'];

animales._lastIndexOf('perro');

>> 3

animales._lastIndexOf('tigre'));

>> 1
