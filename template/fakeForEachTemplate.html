El método fakeForEach()ejecuta la función indicada una vez por cada elemento del array.

Sintaxis

fakeForEach(array, callback)

Parámetros

array
El array sobre cuyos elementos se comprueba si cumplen con la condición provista por la función.
callback
Función que se va a aplicar a los elementos hasta encontrar alguno que la cumpla.

Valor de retorno

Devuelve un elemento del array o undefined en caso de que ninguno cumpla con la condición.

Ejemplo

Mostrar cada elemento del array

                    
let array = ['a', 'b', 'c'];

fakeForEach(array, function(element){          
console.log(element);       
})          
// output: "a"
// output: "b"
